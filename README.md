#  YouTube Data Dashboard

An interactive data analytics dashboard built with Streamlit that provides real-time insights into YouTube channel performance, including video views, watch time, subscriber growth, and engagement metrics.

##  Features

###  Real-time Analytics
- Live channel statistics (subscribers, total views, video count)
- Dynamic video performance metrics
- Real-time data fetching from YouTube API

###  Interactive Visualizations
- **Bar Charts**: Top-performing videos by views, likes, and comments
- **Line Graphs**: Subscriber growth trends over time
- **Scatter Plots**: Views vs engagement rate analysis
- **Engagement Analysis**: Like-to-view and comment-to-view ratios

###  Dynamic Filtering
- Date range selection for time-based analysis
- Minimum views filter
- Top N videos selector
- Customizable analysis sections

###  Engagement Metrics
- Engagement rate calculation
- Like and comment analysis
- Engagement trends over time
- Day-wise engagement patterns

### Watch Time Analytics
- Total watch time calculation
- Average watch time per video
- Optimal video duration insights

###  Data Export
- CSV export functionality
- Comprehensive data with all metrics
- Timestamped exports for version control

##  Technologies Used

- **Streamlit** - Web application framework
- **YouTube Data API v3** - Data source
- **Plotly** - Interactive visualizations
- **Pandas** - Data manipulation and analysis
- **Google API Client** - YouTube API integration

##  Prerequisites

- Python 3.7 or higher
- Google Cloud Platform account
- YouTube Data API v3 enabled
- Valid API key from Google Cloud Console
 
 ## How to Use

Enter API Key and Channel ID

Paste your YouTube API key in the sidebar

Enter the YouTube channel ID (e.g., UC_x5XG1OV2P6uZZ5FSM9Ttw)

Configure Analysis Settings

Select number of videos to analyze (10-100)

Set date range filters

Adjust minimum views threshold

Explore Visualizations

View top-performing videos in bar charts

Analyze subscriber growth trends

Explore engagement metrics

Check watch time analytics

Export Data

Click download button to export as CSV
